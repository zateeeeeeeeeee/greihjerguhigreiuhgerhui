Roblox FastFlags

This page lists all existing Roblox FastFlags, providing their name, type, description, and current value. FastFlags are internal variables used by Roblox to control various features and behaviors.

Type Descriptions

Flag: A boolean variable that can be either true or false.

Int: An integer variable used to store whole numbers.

String: A variable used to store a sequence of characters.

Prefix Descriptions

F (Fast): A regular fastvariable that is initialized once and does not change until a new session begins.

DF (Dynamic Fast): A fastvariable that can change at run-time, and automatically updates every 5 minutes.
